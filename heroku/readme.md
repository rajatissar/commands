# Heroku

```sh
$ heroku login

$ heroku git:remote -a repository_name

$ git push heroku master
    OR
$ git push heroku branch_name:master

$ node --version

$ npm --version

$ git --version

$ heroku create

$ git remote -v

$ heroku logs

$ heroku git:clone -a heroku_branch_name

$ git remote add heroku https://git.heroku.com/heroku_branch_name.git
```
